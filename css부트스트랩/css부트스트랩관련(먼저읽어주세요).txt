일단 저도 기본파일인 style_1.css안에 제가 만든 css는 따로 뺴놓고 style_park.css로 업로드할 예정입니다(충돌 방지용과 나중에 취합할때 편하라고)

드롭박스나 text박스 같은 입력하는 창의 폰트 색상 
#72728c


버튼 기본이고 만약 사이즈가 너무 크거나 작으면 각각 height를 조절해주시면 됩니다(다만 취소,저장,검색 같은건 공통으로 하고
예를들면 회원가입창/아이디찾기/비밀번호찾기/상품등록 등등 안에 아이디/비밀번호나 주소찾기 같은건 버튼높이가 큰 관계로 수정필요
저는 숙소상품 등록에 주소찾기버튼을 color:#0b5394(팀 색상) height: 35px로 줬습니다.)
폭도 너무 길경우 맞춰서 수정필요합니다(admin 파일 쪽에서 높이는 그대로지만 틀에 맞춰서 폭을 줄였습니다)
그런데 기본적으로 class 값은 수정하면 안됩니다(부트스트랩 적용안됨) id값이나 각각 style로 줄수밖에 없습니다

로그인/로그아웃/검색버튼 (기본사이즈 저희가 정한 팀 색상 헤더 상단 우측에 위치한 버튼)
id나 class이름 {
  min-width: 150px;
  height: 52px;
  background-color: #0b5394;
  border-radius: 0;
  padding: 0 15px;
  line-height: 50px;
  color: #fff;
  font-weight: 600;
  font-size: 15px;
  -webkit-transition-duration: 500ms;
  transition-duration: 500ms;
border: none;
}
